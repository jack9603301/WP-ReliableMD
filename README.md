# WP-ReliableMD

## 简介

也许这是最好的wordpress编辑器，为了替换怀城一只猫的编辑器而开发，成为下一代markdown编辑器 

编辑器正在开发中，未正式发布，不建议使用

## 特性

- [x] 基础Markdown
- [ ] 支持公式渲染
  - [x] Katex
  - [ ] MathJax
- [x] 前端+后端，二次渲染支持
- [ ] 可扩展 定制主题
  - [ ] WP Editor.md
  - [ ] WP HyperMD
- [ ] 支持UML
  - [ ] 支持mermaid
  - [x] 支持自带的UML绘制
- [ ] 插件兼容
- [x] 支持图标绘制
- [x] 表格扩展
- [x] 支持shortcode解析markdown
